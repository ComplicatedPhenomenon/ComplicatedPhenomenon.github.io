Data Visualization

Posted on 2018-03-28 | Edited on 2019-11-12 | In Technology , Data Visualization | | Views:
Data visualization is a way of showing data in an impressive way, moreover, you will form an instinct about the data.


What to do with so many options?

Data visualization in web is much or less connected with front-end technology, as a person aims to be a data scientist, they may have poor knowledge in it. Moreover, studying the front-end technology from scratch is not wise, learning the basic syntax of html, css and javascript is extremely boring and not helpful in some sense. Therefore, I determined to make use of the available JS library.

Why? because it’s much more generic, implementing your very particular is theoretically possible once you are a master in the lower level knowledge. Why? well, you can see loads of frameworks but you don’t see many innovative programming languages, similarly, you can use a specific software quite well but you won’t know how to make a cool software just like this one, for that reason you needs to learn the groundwork of front-end technology.

So why learning to use a library is necessary? Well, if you do everything from scratch, you won’t be able to fly? what does this supposed to mean? it means you can do everything from the very beginning, you will be able to travel at the speed of riding a bicycle, how about traveling at the speed of a car?, how about at the speed of a flight? You need to rely on what’s available at present time.

Summary on the relevant JS library

Most data visualization tool are based on d3.js, including vega.js and plotly.js. plotly API library is used to interact with plotly.js.

  • d3 is best for: Hardcore charting experts not afraid to code.

    d3 is a little bit tricky, it takes lots of time for little learning. In other words, your goal might just be to build an interactive visualization with some well-known charts, which can be tricky too.

  • vega is best for: Developers who want the power of d3 without the complexity of learning it from scratch.

  • plotly.py is a package designed to interact with the plotly.js library in order to allow Python users to create plots in their preferred environment.

Question: what is the best data visualization tool for you?

Before jumping into any specific library, I suggest to have a complete investigation of current popular library, Make sure you start learning the usage of a library meet your requirement, if you are uncertain about what do you need, you better save your time to build a solid foundation of JS.

Plotly

Plotly has mad library API for several libraries, but in which the language the core of Plotly is developed?

Plotly was built using Python and the Django framework, with a front end using JavaScript and the visualization library D3.js, HTML and CSS. Files are hosted on Amazon S3.
