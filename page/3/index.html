ROOT

Posted on 2018-06-15 | Edited on 2019-11-12 | In Technology , Code in Physics |
A modular scientific software framework. It provides all the functionalities needed to deal with big data processing, statistical analysis, visualization and storage. It is mainly written in C++ but integrated with other languages such as Python and R.

ROOT is an object-oriented framework aimed at solving the data analysis challenges of high-energy physics. programming inside a framework enables you to care about import problem in your domain, you don’t need to worry about the infrastructure and how they are connected together, the framework leaves you with interface, you do in their way and it works. Like you need light to watch books in the dark, you don’t need to connect the light to your family’s electric route, and your family’s electric comes from electrical factory, how electric produced their, what kind of infrastructure do they need, these thing is transparent to you and you don’t don’t care, someone take care of it, what you need is switch on the light and you get what you want.

A data science platform is a software hub among which all data science work takes place. Clearly, Anaconda is similar with MICROSOFT R, Wolfram Mathematica ROOT among others.

Why do we need isolated environment for each project?

Say you have multiple projects and they all rely on single package, take Django project as an example, assume each of the projects is using a specific version of Django, how do you install multiple version of a package in the OS?

Obviously, it would be great if each of these projects had an isolated environment where they had only the dependencies and the specific packages they need and the version they need.

Concept of types of environments:

  • Global environment
  • Virtual environment
  • Conda environment
